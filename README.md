Marketing Campaign Performance Dashboard
Built with Power BI & AI-powered Python Script

Project Overview
This project analyzes marketing campaign performance using synthetic data generated with Python and visualized using Power BI.
With AI-supported data creation (using the faker library), the goal was to simulate real-world marketing scenarios and visualize performance metrics in an interactive dashboard.

Objective
To design a marketing analytics dashboard that helps understand:
Campaign-wise performance
Trends across months
Conversion and CTR metrics
High-performing marketing channels
